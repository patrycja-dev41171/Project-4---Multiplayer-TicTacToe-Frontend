## Run Locally - BACKEND

Clone the project

```bash
  git clone https://github.com/patrycja-dev41171/Project-4---Multiplayer-TicTacToe-Backend.git
```

Go to the project directory

```bash
  cd TicTacToe-backend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Run Locally - FRONTEND
